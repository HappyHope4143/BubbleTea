## 개인정보 수집 및 이용

BubbleTea("본 앱")는 [개발자/회사명]이(가) 개발한 Android 모바일 애플리케이션입니다. 본 개인정보처리방침은 본 앱이 귀하의 정보를 수집, 사용, 공개하는 방법에 대해 설명합니다.

## 수집하는 정보

### 자동으로 수집되는 정보

본 앱은 다음과 같은 정보를 자동으로 수집할 수 있습니다:

1. **기기 정보**
   - 기기 모델 및 제조업체
   - Android 운영체제 버전
   - 고유 기기 식별자
   - 앱 버전 정보

2. **사용 정보**
   - 앱 사용 패턴
   - 화면 조회 횟수
   - 기능 사용 통계
   - 오류 및 충돌 보고서

3. **인터넷 활동**
   - 뉴스 API 요청 (NewsAPI.org를 통해)
   - 네트워크 연결 상태

### 광고 관련 정보

본 앱은 Google AdMob을 사용하여 광고를 표시합니다. AdMob은 다음 정보를 수집할 수 있습니다:

- 광고 ID (AAID)
- IP 주소
- 광고 클릭 및 조회 데이터
- 기기 정보 및 위치 정보 (대략적인 위치)

AdMob의 개인정보처리방침: https://policies.google.com/privacy

### 수집하지 않는 정보

본 앱은 다음 정보를 **수집하지 않습니다**:

- 이름, 이메일, 전화번호 등 개인 식별 정보
- 정확한 위치 정보 (GPS)
- 연락처, 사진, 파일 등 개인 데이터
- 사용자 계정 정보

## 정보의 사용 목적

수집된 정보는 다음 목적으로 사용됩니다:

1. **앱 기능 제공**
   - 기술 뉴스 콘텐츠 제공 (NewsAPI.org 활용)
   - 뉴스 위젯 기능 제공
   - 오프라인 뉴스 캐싱 (Room 데이터베이스)

2. **앱 개선**
   - 버그 및 오류 수정
   - 성능 최적화
   - 사용자 경험 개선

3. **광고 표시**
   - 맞춤형 광고 제공 (Google AdMob)
   - 광고 성과 측정

## 정보의 저장

- **로컬 저장**: 뉴스 데이터는 귀하의 기기에 로컬로 저장됩니다 (Room 데이터베이스).
- **캐시 관리**: 최신 100개의 뉴스만 자동으로 유지되며, 오래된 데이터는 자동으로 삭제됩니다.
- **데이터 암호화**: 네트워크 전송 중 데이터는 HTTPS를 통해 암호화됩니다.

## 제3자 서비스

본 앱은 다음 제3자 서비스를 사용합니다:

1. **NewsAPI.org**
   - 목적: 기술 뉴스 콘텐츠 제공
   - 웹사이트: https://newsapi.org
   - 개인정보처리방침: https://newsapi.org/privacy

2. **Google AdMob**
   - 목적: 광고 표시 및 수익화
   - 웹사이트: https://admob.google.com
   - 개인정보처리방침: https://policies.google.com/privacy

3. **Google Play Services**
   - 목적: Android 플랫폼 기능 활용
   - 개인정보처리방침: https://policies.google.com/privacy

## 귀하의 권리

귀하는 다음과 같은 권리를 가집니다:

1. **정보 열람**: 저장된 로컬 데이터를 확인할 수 있습니다.
2. **정보 삭제**: 앱을 삭제하면 모든 로컬 데이터가 제거됩니다.
3. **광고 설정**: Android 설정에서 광고 추적을 제한할 수 있습니다.
   - 설정 > Google > 광고 > 광고 맞춤설정 선택 해제
